# Scripts Directory

This directory contains utility and test scripts for SignSpeak.

## Files

- `live_test.py` - Simple live gesture testing
- `dynamic_testing.py` - Advanced dynamic gesture testing with TTS
- `dynamic_training.py` - Training data collection utilities
- `view_data.py` - Data visualization utilities
- `2_run_translator.py`, `3_run_translator.py` - Alternative translation scripts

## Usage

These scripts are for development and testing purposes. For production use, see:
- `software/main.py` - Main application
- `ml/inference.py` - Simple inference
